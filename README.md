# VDSina Client

Android-клиент для панели управления хостинг-провайдера [VDSina](https://vdsina.com).

## Описание

Приложение представляет собой нативную обертку над веб-интерфейсом панели управления VDSina с дополнительными функциями:

- Поддержка нескольких профилей (аккаунтов)
- Автоматическое сохранение сессий для каждого профиля
- Функция автологина
- Быстрое переключение между аккаунтами


## Требования

- Android 7.0 (API 24) и выше
- Доступ к интернету

## Сборка

### Требования для сборки

- JDK 17
- Android SDK 35
- Gradle 8.5+

### Команды

```bash
# Сборка debug-версии
./gradlew assembleDebug

# Сборка release-версии
./gradlew assembleRelease
```

APK файлы будут находиться в `app/build/outputs/apk/`


## Использование

### Первый запуск

При первом запуске приложение создаст профиль по умолчанию с URL `https://cp.vdsina.com/`.

### Управление профилями

1. Откройте меню (три точки в правом верхнем углу)
2. Выберите "Профили"
3. Нажмите "+" для добавления нового профиля
4. Укажите:
   - Название профиля
   - URL панели управления (по умолчанию `https://cp.vdsina.com/`)
   - Логин (email)
   - Пароль

### Переключение между профилями

1. Откройте меню -> "Профили"
2. Выберите нужный профиль из списка

При переключении профилей автоматически сохраняются и восстанавливаются cookies, что позволяет поддерживать активные сессии для каждого аккаунта.

## Технологии

- Kotlin
- Android Jetpack (AppCompat, ViewBinding)
- Material Design Components
- WebView

## Структура проекта

```
app/src/main/java/com/vdsina/app/
    MainActivity.kt       # Главная активность с WebView
    Profile.kt            # Модель данных профиля
    ProfileManager.kt     # Управление профилями и cookies
    ProfilesActivity.kt   # Экран управления профилями
```

## Безопасность

Учетные данные хранятся локально на устройстве в SharedPreferences. Приложение не передает данные третьим сторонам.
